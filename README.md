# Java File Encryptor

A simple Java program that encrypts and decrypts files using the AES algorithm. I built this following a java tutorial as I am keen to refresh my java knowledge.

## Features
- Encrypts and decrypts any file
- Uses AES symmetric encryption
- Console-based, lightweight, and easy to run
